TA Office Hours Scheduler

Assigns TA offers hours to their preferences, which are obtained via Google Forms.

  • Adjustable Office Hours Times
  • Multiple TAs per office hours slot
  • "Scaling" preferences (e.g. preferring office hours just before homework deadlines)
  • Exports final schedule to an .ics file (compatible with most calendar apps)

Instructions

  • TAs provide their office hours preferences by filling out a Google Form (or download our example to test it right now)
  • You can edit the office hours timeslots directly on your own google form (e.g. swap "8AM-9AM" to "10:30AM-10:45AM") documentation
  • Configuration parameters are passed via a yaml file. Please see configuration documentation for details (and a default configuration template)

Data Collection and Storage

To collect usage stats, each run generates a timestamp and a list TA email hashes which can't be used to identify the original email address. For example:

2025-04-22 13:22:05,489 - INFO - RUN 42038f0a,83387ad1,2840a8f3,1383079d, ...
